Features

What OpenRelay does

A compact always-on-top assistant window designed for quick prompts, short research, and on-demand image generation.

Streaming chat

Assistant responses stream into the output area as they arrive, with a lightweight shimmer-style loading state before the first token appears.

Image generation flow

When you explicitly ask for an image, the app can switch from chat output into image generation and still fall back if the model answers with a refusal instead of the image tag.

Compact popup UI

The interface is frameless, draggable, always on top, and remembers its last size and position so it opens like a tool instead of a full app window.

Model selection

Text and image models are selectable from the top bar, including Gemini, Claude, GPT, DeepSeek, flux, zimage, and flux-2-dev options.

Built-in image actions

Generated images can be opened in a larger overlay, copied to the clipboard, or saved locally without leaving the popup.

Quick keyboard access

With keyboard support enabled, Ctrl + Space brings the popup forward instantly for fast prompts and short back-and-forth use.

How it works

Simple popup workflow

  1. Open the popup and type into the single input field.
  2. Watch the reply stream in as the model responds.
  3. Ask for an image when you want generation instead of text.
  4. Use the footer actions to copy text, clear context, open images, copy them, or save them.

Run locally

Install

pip install PyQt6 keyboard tiktoken
python research.py
